# Welcome to your Expo app 👋

This is an [Expo](https://expo.dev) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## Get started

1. Install dependencies by going to the root directory and running `bun install`

2. Ensure that you have a .env file in the root of the example directory with the following contents:

   ```
    EXPO_PUBLIC_SPOTIFY_CLIENT_ID=your-spotify-client-id
    EXPO_PUBLIC_SPOTIFY_REDIRECT_URL=example://spotify-auth-callback
   ```

2. Start the app by going to the root directory and running `bun run example`
